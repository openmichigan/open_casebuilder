Create case elements using the tabs in the left pane. Click the Add button then watch the case develop in the Preview box. Before you exit this page, click Export to view the complete, stylized case in another tab:
  1. Click Export.
  2. In the new tab select File > Save Page As.
  3. Select Format = Webpage, Complete.
  4. Click the Save button. This will create a .html file and a corresponding folder with the same name (eg. case_one.html and a folder called case_one).
  5. In order to share your case with others, create a zip file that contains the .html file and the folder (do not move the .html file within the case folder, it must be at the same level as the folder), then distribute to others online or via removeable media (e.g. USB drives).
