Create and add case elements using the tabs in the right pane and the Add button. Watch the case develop in the Layout box. Before you exit this page, click Preview and Export to view the complete, stylized case in another tab:
  1. Click Preview & Export.
  2. Review the output. If necessary, return to this tab to make edits, and then click Preview & Export to view the updates.
  3. In the new tab select File > Save Page As. (see screenshot).
  4. Select Format = Webpage, Complete. (see screenshot)
  5. Click the Save button. This will create a .html file and a corresponding folder with the same name (e.g. CaseA.html and a folder called CaseA_files).
  6. Create a zip file that contains the .html file and the folder. (The .html file must be at the same level as - not within - the case folder.) (see screenshot)
  7. Distribute to others online or offline (e.g. DVD, USB drives).
