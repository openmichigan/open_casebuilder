Open Case Builder is a downloadable web app that can be used to create instructional modules such as patient cases, scenarios, quizzes, or readings. Simply fill out the forms to insert text, media, and questions and use the arrows to arrange them in the desired order. No Internet connection nor knowledge of web page design (HTML) is required.

 

Click Create Case to begin. Be sure to preview and export your case before you exit the Create Case page, as the current version of Open Case Builder does not allow authors to import or return to previous cases.


NOTE: This application is designed to work with the Google Chrome free web browser. Other web browsers may not support full functionality for Open Case Builder. Due to some restrictions with the HTML5 format regarding presentation of audio and video, we recommend that both authors and learners use Google Chrome to create and view the cases that integrate audio or video.


Release: Open Case Builder v 1.2. Last revised April 9, 2013.
